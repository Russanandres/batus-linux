# BatusLin.sh
this is the sh where the standard bash commands for Ubuntu are collected in one place

Russian only version now

## Installing and running
Before run check executing permissions.
Type in terminal:

$: ```sudo chmod +x BatusLin.sh```

$: ```./BatusLin.sh```
